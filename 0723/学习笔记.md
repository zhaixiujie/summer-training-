# 一、python 运行程序方式

## 命令行运行

**Python文件的完整路径**
* cmd 中运行 win+r 输入cmd
* 在Python文件中按住shift键邮件打开Windows powersheell窗口，输入python xx.py即可运行
```bash
python xx.py
```

## idle运行
* Ctrl+s 保存之后f5直接运行 

## vscode运行

* 右键选择`在终端中运行Python文件

* 安装code-runner插件，然后点击右上方三角形

# 二、Python注释方式
* 单行注释：#加上注释的内容

```bash
# HELLO WORLD 这是一个单行注释
print("hello world")
```
* 多行注释：用的是一对```三个单引号构成或者是一对"""三个双引号构成

```bash
'''
这是一个多行注释
print（1）
'''
```
# 三、web注释方式
* <%-- xxxxx --%>
```bash
<%--    <input type="button" value="这是一个按钮" />--%>
```
# 四、web中一些代码意义
* < br/>表示换行
* &lt 在浏览器中为<>
* < span xxxxx> : 限制一行
